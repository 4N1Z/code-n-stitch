# Paint App
Paint Application

### Prerequisites
```PyQT5``` Module
* Install it by running: ```pip3 install PyQt5```

### How to run the script
First you have to go to the Paint Application directory. Run the following command once you are in project directory

For Python 3: ```python3 main.py```
* Then start drawing whatever you want. You can change brush sizes and colors too. Under Files tab you can save your drawing as a ```.jpeg``` or ```.png```
file.
